# FORC Literature References

This directory contains key literature references for the First-Order Reversal Curve (FORC) implementation:

   * **Journal**: *Journal of Applied Physics*, Vol. 85, No. 9, pp. 6660–6667 (1999)

   * **Journal**: *Journal of Geophysical Research: Solid Earth*, Vol. 105, No. B12, pp. 28461–28475 (2000)
